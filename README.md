# Weather App

A comprehensive weather application providing current conditions, 48-hour forecasts, and 5-day outlooks for any location worldwide.

## features

1) Search By City : Find weather data for any city globally
2) Geolocation : Automatic weather detection for your current location
3) Toggle between Celsius and Fahreneit for the main temperature
4) 48-Hour Forecast : Shows detailed forecast for 48 hours 
5) 5-Day Forecast : Detailed forecast with high/low temperatures and dates
6) Dynamic Backgrounds : Display a background based on the weather
7) Responsive Design : Desktop,Tablet and mobile friendly
8) Smooth Animations

## Technologies Used

HTML5  -- structure and semantic
CSS3 -- used flexbox,grid and animations
JavaScript -- DOM manipulation,real world updates with setInterval and fetch data from an API
OpenWeatherMAP API -- real time weather data
Geolocation API -- Browsed based location detection

## API Key Setup

This app requires a free OpenWeatherMap API key which is free

## Steps

1. Sign up at [text](https://home.openweathermap.org/users/sign_in) to get the API key
2. Copy API and paste inside the config-test.js 
3. Open index.html and the app should work !
